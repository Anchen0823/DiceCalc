# DiceCalc

博德之门3 风格骰子检定概率计算器。

## 简介

基于 1d20 的检定概率计算：**自然 1 视为大失败（必败）**，**自然 20 视为大成功（必胜）**，其余 2–19 与加值之和 ≥ 检定难度（DC）则成功。

## 使用说明

1. 用浏览器打开项目根目录下的 **`index.html`** 即可使用（无需服务器）。
2. **检定难度 DC**：输入目标值（1–30），例如 15 表示需要 ≥15 才成功。
3. **加值（可选）**：如技能加值、属性调整值等，支持负数。
4. 页面会实时显示：成功概率、大成功/大失败各 5%，以及普通成功/普通失败占比。

## 规则简述

- **1** → 大失败，必定失败  
- **20** → 大成功，必定成功  
- **2–19** → 骰面 + 加值 ≥ DC 则成功，否则失败  

## 文件说明

- `index.html` — 单页图形界面与计算逻辑（开箱即用）
